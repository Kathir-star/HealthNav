# Contributing to HealthNav

We welcome contributions to HealthNav! By participating in this project, you agree to abide by our [Code of Conduct](CODE_OF_CONDUCT.md).

## How to Contribute

### 1. Fork the Repository

Fork the [HealthNav repository](https://github.com/Kathir-star/HealthNav) to your own GitHub account.

### 2. Clone Your Fork

```bash
git clone https://github.com/YOUR_USERNAME/HealthNav.git
cd HealthNav
```

### 3. Create a New Branch

Create a new branch for your feature or bug fix:

```bash
git checkout -b feature/your-feature-name
# or
git checkout -b bugfix/issue-description
```

### 4. Make Your Changes

*   Ensure your code adheres to the project's coding style.
*   Write clear, concise, and well-documented code.
*   Add or update tests as appropriate for your changes.

### 5. Test Your Changes

Run the tests to ensure everything is working correctly:

```bash
npm test
# or
yarn test
```

### 6. Commit Your Changes

Commit your changes with a descriptive commit message. Follow conventional commit guidelines if applicable.

```bash
git commit -m "feat: Add new feature X"
# or
git commit -m "fix: Resolve issue Y"
```

### 7. Push to Your Fork

```bash
git push origin feature/your-feature-name
```

### 8. Create a Pull Request

1.  Go to the original [HealthNav repository](https://github.com/Kathir-star/HealthNav).
2.  Click on the "New Pull Request" button.
3.  Ensure your branch is selected and provide a clear title and description for your pull request.
4.  Link any relevant issues.

## Pull Request Guidelines

*   **One feature/fix per PR**: Keep your pull requests focused on a single change.
*   **Descriptive title**: Clearly state what the PR does.
*   **Detailed description**: Explain the problem your PR solves, how it solves it, and any potential side effects.
*   **Tests**: Include tests that cover your changes.
*   **Code Style**: Ensure your code matches the existing style.
*   **Review**: Be responsive to feedback during the review process.

## Code of Conduct

By contributing, you are expected to uphold our [Code of Conduct](CODE_OF_CONDUCT.md).

Thank you for contributing to HealthNav!
